# 易考自动配置网页 - 同事操作指导文档

本文档用于指导同事从 GitHub 拉取项目，并在本机启动易考自动配置网页。

## 1. 项目地址

GitHub 仓库：

```text
https://github.com/atachenjun-cm/yikao-auto-config-web
```

## 2. 运行环境要求

请先确认本机具备以下环境：

- Node.js 18 或更高版本
- Python 3.10 或更高版本
- Google Chrome
- 可正常访问易考后台

建议先检查版本：

```bash
node -v
python3 --version
```

## 3. 克隆项目

打开终端，执行：

```bash
git clone https://github.com/atachenjun-cm/yikao-auto-config-web.git
cd yikao-auto-config-web
```

## 4. 安装依赖

### 4.1 安装 Node 依赖

```bash
npm install
```

### 4.2 安装 Python 依赖

```bash
python3 -m pip install -r requirements.txt
```

## 5. 配置环境变量

复制模板文件：

```bash
cp .env.example .env
```

`.env` 示例：

```env
PORT=8765
CODEX_PYTHON=/absolute/path/to/python3
```

说明：

- `PORT`：本地网页服务端口，默认 `8765`
- `CODEX_PYTHON`：Python 解释器完整路径

如果不知道 Python 路径，可以执行：

```bash
which python3
```

把结果填入 `CODEX_PYTHON`。

## 6. 启动项目

执行：

```bash
npm start
```

看到服务启动后，在浏览器打开：

```text
http://127.0.0.1:8765
```

## 7. 首次使用流程

进入网页后，按以下步骤操作：

1. 填写易考后台登录信息
2. 导入需求单 Excel
3. 点击开始执行
4. 等待网页自动配置考试
5. 查看执行日志和配置结果

## 8. 登录信息说明

登录地址、账号、密码是在网页内填写的，不要写进代码。

这些运行时信息会保存在本地目录：

```text
.easy_exam_runtime/
```

该目录不会提交到 GitHub。

## 9. 常见问题

### 9.1 网页打不开

先确认服务是否正常启动：

```bash
npm start
```

再确认端口没有被占用。

### 9.2 Python 解析 Excel 失败

先检查：

```bash
python3 -m pip install -r requirements.txt
```

并确认 `.env` 里的 `CODEX_PYTHON` 路径正确。

### 9.3 浏览器自动化没有执行

重点检查：

- Chrome 是否已安装
- 易考后台是否可访问
- 登录地址、账号、密码是否填写正确
- 需求单字段是否完整

### 9.4 运行时报权限或依赖问题

建议重新执行：

```bash
npm install
python3 -m pip install -r requirements.txt
```

## 10. 更新项目

以后如果仓库代码有更新，可在项目目录执行：

```bash
git pull
npm install
python3 -m pip install -r requirements.txt
```

## 11. 安全要求

不要提交以下文件或内容：

- `.env`
- `.easy_exam_runtime/`
- 登录账号密码
- Cookie
- Token

## 12. 标准启动命令汇总

首次安装：

```bash
git clone https://github.com/atachenjun-cm/yikao-auto-config-web.git
cd yikao-auto-config-web
npm install
python3 -m pip install -r requirements.txt
cp .env.example .env
npm start
```

日常启动：

```bash
cd yikao-auto-config-web
npm start
```
